# Anime-Next-Watch-Generator
A smart anime recommendation tool that suggests what to watch next based on user preferences, genres, and watch history.
## 📌 Overview
Anime Next Watch Generator is a simple and interactive web application that helps users discover what anime to watch next. It provides recommendations based on selected genres and user preferences using a clean UI and JavaScript logic.

## 🚀 Features
- 🎯 Anime recommendations based on genre  
- 🎨 Clean and responsive user interface  
- ⚡ Fast and lightweight (no frameworks)  
- 🔍 Easy selection of preferences  
- 📱 Works directly in the browser  

## 🛠️ Tech Stack
- HTML5 – Structure of the web app  
- CSS3 – Styling and responsiveness  
- JavaScript (Vanilla JS) – Logic and interactivity
